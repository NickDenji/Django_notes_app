# Django_notes_app
Sticky Notes Application

Overview

This is a web-based sticky notes application built using Django.

The application follows Django’s MTV (Model-Template-View) architecture.

Features:

Create new notes
View all notes
View individual note details
Edit existing notes
Delete notes
Styled user interface using CSS
Technologies Used:

Python
Django
SQLite
HTML
CSS
Project Structure:

sticky_notes/ - main Django project configuration
notes/ - application containing models, views, forms, and URLs
templates/notes/ - HTML templates
static/css/styles.css - styling
manage.py - Django management script
